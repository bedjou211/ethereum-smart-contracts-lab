# ethereum-smart-contracts-lab

## Devine Nombre – DApp Ethereum avec Solidity, Remix et web3.py

#### 2. Description courte

Projet de développement d’une application décentralisée autour d’un jeu “Devine Nombre”, implémenté avec Solidity pour le smart contract et Python/web3.py pour l’interaction avec la blockchain Ethereum.

#### 3. Objectifs
comprendre le fonctionnement des smart contracts
développer et déployer un contrat Solidity
interagir avec la blockchain via web3.py
concevoir une DApp simple avec logique métier on-chain
#### 4. Fonctionnalités
création d’une partie
choix d’un nombre secret
tentatives limitées pour deviner le nombre
réponse du contrat : plus grand / plus petit / correct
éventuellement gestion d’une mise en ETH
version avancée avec confidentialité du nombre
#### 5. Stack technique
Solidity
Remix IDE
Python
web3.py
Ethereum / Sepolia testnet
